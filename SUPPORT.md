# Support

## Getting help

- Usage/setup questions: open a GitHub Discussion or Issue.
- Bug reports: use the bug template in `.github/ISSUE_TEMPLATE/bug_report.md`.
- Feature proposals: use the feature template in `.github/ISSUE_TEMPLATE/feature_request.md`.

## Security issues

Do not open public issues for vulnerabilities.

Report via GitHub Security Advisories as documented in [SECURITY.md](SECURITY.md).

## Scope

Maintainers provide best-effort support for:

- Tool contract behavior
- Reproducible bugs
- Documentation gaps

Out of scope:

- Private dataset licensing decisions
- Custom downstream product integrations
- Real-time performance scoring (not in v1)
